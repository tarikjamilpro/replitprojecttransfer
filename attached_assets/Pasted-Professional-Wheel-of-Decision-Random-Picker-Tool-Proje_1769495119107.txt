Professional Wheel of Decision - Random Picker Tool
Project Overview
Build a modern, engaging Wheel of Decision (Random Picker) web application using Next.js 16, React 19, and TypeScript. This fun and interactive tool helps users make decisions by randomly selecting from a list of options with exciting animations and celebratory effects.

Core Features
1. Options Input System
Input Text Area:

Large textarea for entering options
Placeholder: "Enter your options here... (one per line)\n\nExample:\nPizza\nBurger\nBiryani\nSushi\nPasta"
Minimum height: 200px
Auto-expanding based on content
Line counter: "X options entered"
Clear all button
Input Validation:

Minimum 2 options required to spin
Maximum 50 options (configurable)
Remove empty lines automatically
Trim whitespace from each option
Show error if insufficient options
Quick Actions:

"Add Option" button (alternative to typing)
"Clear All" button
Pre-filled example options (demo mode)
Import from file option (advanced)
2. Spin Animation System
Spin Button:

Large, prominent button
Text: "Spin the Wheel!" or "Pick for Me!"
Icon: Shuffle, dice, or wheel icon (lucide-react)
Disabled states:

When spinning (prevent multiple spins)
When less than 2 options
Loading animation while spinning
Animation Behavior:

Phase 1: Preparation (200ms)

Show all options in a grid or list
Brief highlight effect on all options
Phase 2: Rapid Selection (2000-3000ms)

Rapidly cycle through options with highlighting
Speed starts fast, gradually slows down
Each option highlights for 50-100ms initially
Slow down to 200-300ms near the end
Visual: Scale, color change, or glow effect
Phase 3: Final Selection (500ms)

Slow down dramatically
Highlight winner with emphasis
Hold on winner for dramatic effect
