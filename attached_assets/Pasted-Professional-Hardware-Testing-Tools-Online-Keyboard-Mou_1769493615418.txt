Important Notes
Event Handling
Use event.code not event.key:

event.code represents physical key location (KeyA, Digit1)
event.key represents character produced (changes with Shift, Caps)
Prevent default behavior: Especially for Tab, Backspace, etc.
Handle modifier keys: Left/Right Shift, Ctrl, Alt, Meta separately
