Build a Clean and Professional Text-to-Image Generation Web App Using Bytez SDK

Prompt Description:
Develop a modern, professional-grade web application on Replit that allows users to generate images from text input using the Bytez API and its Dreamlike Photoreal 2.0 model. The UI should be user-friendly, responsive, and visually appealing. It should offer a smooth user experience for generating high-quality images from descriptive text prompts.

Key Features and Requirements:

User Interface:

A clean input area to enter descriptive text prompts for image generation.
A clear “Generate Image” button to trigger image creation.
Display the generated image result instantly below or beside the input.
Show loading animations or progress indicators during image generation.
Responsive design suitable for desktop and mobile devices.
Functionality:

Integrate the Bytez Python SDK with API key securely handled in backend environment variables.
Use the dreamlike-art/dreamlike-photoreal-2.0 model from Bytez SDK for generating photorealistic images from the user prompt.
Retrieve and display any error messages returned by the Bytez SDK gracefully on the UI.
Optimize input validation (non-empty, sensible length limit, e.g., 5-100 words).
Support refreshing or clearing the input to generate new images easily.
Technical Specs:

Backend: Python Flask or FastAPI API within Replit environment to run Bytez SDK requests securely.
Frontend: JavaScript/React or plain HTML + JavaScript + CSS with fetch/AJAX to call backend API endpoint.
Environment variables storage in Replit for BYTEZ_KEY.
Error handling for network/API issues.
Optional: Allow user to download generated images.
User Experience Enhancements:

Add placeholders and helper text on input box like “Enter your image description here...”
Provide guidance about suitable prompt examples.
Visual feedback upon successful image generation or failure.
Optionally add a gallery of recent images generated by the user session.
