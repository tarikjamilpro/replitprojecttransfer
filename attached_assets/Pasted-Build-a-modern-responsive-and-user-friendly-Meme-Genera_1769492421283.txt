Build a modern, responsive, and user-friendly Meme Generator web application using Next.js 16 and React 19 with TypeScript.

Core Requirements
1. Image Handling System
Provide users with two options:

Option A: Upload custom images (support drag & drop + file picker)
Option B: Select from 5-10 popular meme templates stored in /public/memes/ folder

2. Text Input Interface
Implement two clear, accessible input fields:

"Top Text" input field
"Bottom Text" input field
Requirements:

Real-time preview: text appears instantly on image as user types
Input validation with character limit (max 100 characters per field)
Responsive and mobile-friendly
3. Text Styling & Rendering
Typography specifications:

Font: Impact or similar bold meme-style font
Color: Pure white (#FFFFFF)
Outline: 2-3px black (#000000) stroke for readability on any background
Position:

Top text: Centered in upper portion of image
Bottom text: Centered in lower portion of image
Responsive: Text size auto-adjusts based on image dimensions
4. Canvas-based Image Processing
Use HTML5 Canvas API to render image + text
Ensure high-quality output (minimum 800px width)
Apply proper anti-aliasing for smooth text rendering
Maintain aspect ratio of original images
5. Download Functionality
Implement a prominent "Download Meme" button:

Export format: PNG (lossless quality)
Filename pattern: meme-${timestamp}.png
Preserve original image quality
Handle download errors gracefully
