 Professional CPS Test (Click Speed Test) Application ### Project Requirements Create a modern, interactive **Click Speed Test (CPS Test)** gaming tool using Next.js 16, React 19, and TypeScript. This application will measure user clicking speed with accurate timing, visual feedback, ranking system, and social sharing capabilities. --- ### Core Features #### 1. Time Duration Selector - Display 5 buttons for test duration selection: **1s, 5s, 10s, 30s, 60s** - Default selection: **10 seconds** - Use button group layout (horizontal on desktop, responsive on mobile) - Highlight selected duration with primary color - Disable during active test #### 2. Main Click Area **Before Test:** - Large centered button (300x300px minimum) - Text: "Click Here to Start" - Eye-catching design with border and subtle shadow - Pulsing animation to attract attention **During Test:** - Text changes to current click count or "Keep Clicking!" - Visual feedback on each click (ripple effect, scale animation) - Display countdown timer (updating in real-time) - Show current click count prominently **After Test:** - Disable clicking - Smooth transition to results screen #### 3. Game Logic **Test Flow:** - **Phase 1 (Setup):** User selects duration, sees "Click Here to Start" - **Phase 2 (Testing):** First click starts timer, count all clicks until time expires - **Phase 3 (Results):** Display CPS score, rank, and sharing options **Timing Implementation:** - Use precise timing with `performance.now()` or `Date.now()` - Update countdown every 50-100ms for smooth display - Stop accepting clicks immediately when timer reaches 0 - Calculate exact CPS: `Total Clicks / Duration (seconds)` #### 4. CPS Calculation & Display - Formula: **CPS = Total Clicks ÷ Test Duration** - Display CPS with 1 decimal place (e.g., 8.5 CPS) - Show in extra large, bold typography - Use primary/accent color for emphasis #### 5. Ranking System Implement 7-tier ranking based on CPS score: | CPS Range | Rank | Emoji | |-----------|------|-------| | 0.0 - 3.0 | Turtle | 🐢 | | 3.1 - 5.0 | Rabbit | 🐇 | | 5.1 - 7.0 | Fast Fingers | ⚡ | | 7.1 - 9.0 | Speed Demon | 🔥 | | 9.1 - 11.0 | Pro Gamer | 💪 | | 11.1 - 13.0 | Elite Clicker | 🏆 | | 13.1+ | God Mode | 👑 | Display rank prominently with emoji, title, and themed color. #### 6. Results Screen Show after test completion: - **CPS Score** (large, bold, primary color) - **Rank Badge** (with emoji and title) - **Total Clicks** (with icon) - **Test Duration** (confirmation of selected time) - **Try Again Button** (restart test) - **Share Score Button** (copy to clipboard) #### 7. Share Functionality **Share Button:** - Text: "Share Score" with share icon - Click action: Copy formatted text to clipboard - Show success toast: "Copied to clipboard!" **Share Text Format:** ``` 🎮 I scored [X.X] CPS on the Click Speed Test! 🏆 Rank: [Rank Name] Can you beat me? Try it now! ``` Use `navigator.clipboard.writeText()` with fallback for older browsers. --- ### Technical Requirements **Tech Stack:** - Next.js 16 (App Router) - React 19 with TypeScript - Tailwind CSS v4 - shadcn/ui components (Button, Card, Badge) - lucide-react icons **File Structure:** ``` /app /page.tsx # Main CPS test page /layout.tsx # Root layout /globals.css # Tailwind styles /components /cps-test.tsx # Main test component /time-selector.tsx # Duration buttons /click-area.tsx # Clicking zone /results-display.tsx # Results screen /ui # shadcn components /lib /utils.ts # Utilities /cps-calculator.ts # CPS logic /ranking.ts # Rank determination ``` **State Management:** ```typescript interface CPSTestState { phase: 'setup' | 'testing' | 'results'; selectedDuration: number; clickCount: number; timeRemaining: number; cps: number; rank: string; } ``` --- ### Design Guidelines **Color Scheme:** - Choose 3-5 colors total (1 primary brand color + 2-3 neutrals + 1-2 accents) - Use vibrant gaming colors (electric blue, neon green, or energetic purple) - Different color for each rank tier - High contrast for click area **Typography:** - Maximum 2 font families - CPS Score: Extra large, bold (text-6xl) - Click Counter: Large (text-4xl) - Rank Title: Bold, uppercase (text-2xl) **Layout:** - Use Flexbox for most layouts - Center-aligned click area - Responsive design (mobile-first) - Card-based sections - Smooth transitions between phases **Visual Effects:** - Ripple effect on each click - Scale animation (button shrinks slightly) - Color flash feedback - Countdown urgency (red color when <3 seconds) - Results fade-in animation --- ### Implementation Priority **Phase 1: Core Functionality** 1. Create page structure with time selector 2. Build click area component 3. Implement click counting 4. Add timer countdown logic 5. Calculate and display CPS **Phase 2: Ranking & Results** 1. Implement 7-tier ranking system 2. Create results display component 3. Add retry functionality 4. Visual feedback and animations **Phase 3: Sharing** 1. Add share button 2. Implement clipboard copy 3. Show success toast notification **Phase 4: Polish** 1. Responsive design 2. Smooth animations 3. Accessibility improvements 4. Performance optimization --- ### Key Implementation Details **Timer Logic:** ```typescript // Start test on first click const startTest = () => { const startTime = Date.now(); const endTime = startTime + (selectedDuration * 1000); const interval = setInterval(() => { const remaining = Math.max(0, endTime - Date.now()); setTimeRemaining(remaining / 1000); if (remaining <= 0) { clearInterval(interval); endTest(); } }, 50); }; ``` **Click Handler:** ```typescript const handleClick = () => { if (phase === 'setup') { startTest(); setClickCount(1); } else if (phase === 'testing') { setClickCount(prev => prev + 1); } }; ``` **Rank Calculation:** ```typescript function calculateRank(cps: number): string { if (cps <= 3.0) return 'Turtle'; if (cps <= 5.0) return 'Rabbit'; if (cps <= 7.0) return 'Fast Fingers'; if (cps <= 9.0) return 'Speed Demon'; if (cps <= 11.0) return 'Pro Gamer'; if (cps <= 13.0) return 'Elite Clicker'; return 'God Mode'; } ``` --- ### Success Criteria - Test starts on first click - Click counter updates without lag - Timer counts down precisely to 0 - Clicking disabled when timer ends - CPS calculated accurately - Rank assigned correctly - Results displayed prominently - Share button copies to clipboard - Toast notification shows success - Responsive on all devices - Smooth animations throughout --- ### Additional Features (Optional) - Save high scores to localStorage - Display personal best indicator - Sound effects on clicks - Confetti animation for high scores - Statistics tracking over time - Dark/light mode toggle --- ### User Instructions Display helpful text on the page: **How to Play:** 1. Select your test duration (1-60 seconds) 2. Click the button to start 3. Click as fast as you can! 4. Check your CPS score and rank 5. Share your results **Tips:** - Use one or two fingers - Find a comfortable rhythm - Try different durations --- **Expected Deliverable:** A fully functional, production-ready CPS Test application with accurate timing, visual feedback, 7-tier ranking system, sharing functionality, and excellent user experience across all devices.
