Build a modern, intuitive Random Color Palette Generator web application using Next.js 16, React 19, and TypeScript. This tool helps designers discover harmonious color combinations with interactive features for locking colors and copying hex codes.

2. Color Generation Logic
Harmonious Color Algorithm: Use HSL (Hue, Saturation, Lightness) for better color harmony.

Color Scheme Strategies (Randomly select one):

Analogous: Colors near each other on color wheel

Base hue ± 30-60 degrees
Complementary: Opposite colors on color wheel

Base hue and hue + 180 degrees
Triadic: Three evenly spaced colors

Base hue, hue + 120°, hue + 240°
Monochromatic: Same hue, varying saturation/lightness

Same hue with different S and L values
Random Balanced: Random hues with consistent saturation/lightness ranges

Design Guidelines
Color Bar Design:

Full height: 100vh or min-height: 600px
Equal width: 20% each (flex: 1)
No gaps between bars (seamless)
Smooth color transitions (transition: background-color 300ms ease)
Text Contrast:

Calculate luminance of background color
Use white text on dark backgrounds (luminance < 0.5)
Use black text on light backgrounds (luminance ≥ 0.5)
Ensure WCAG AA contrast ratio (4.5:1 minimum)
Typography:

Hex code: Large, bold, monospace font (text-3xl or text-4xl)
"Click to copy" hint: Smaller, lighter (text-sm, opacity-70)
Font family: Monospace for hex codes, sans-serif for UI
Interactions:

Hover effects on all interactive elements
Smooth transitions (200-300ms)
Scale animation on click (scale: 0.95)
Cursor changes: pointer for clickable areas
Responsive Design:

Desktop: 5 vertical bars side by side
Tablet: Consider 5 bars or adjust bar width
Mobile: Stack vertically OR horizontal scroll OR smaller bars
Animations:

Color transition: Fade between colors (300ms)
Lock toggle: Rotate or scale animation
Copy feedback: Brief scale + opacity change
Generate button: Rotate icon on click
